MRS

Analysis of data from MR spectroscopy experiments.

Documentation

Installation

To install run:

python setup.py install

Usage

mrs-analysis.py: This is a little command-line utility that takes nifti files as input and produces a csv file as output, with the ppm scale and the spectra from the two measurement echos. See mrs-analysis.py --help for help.
